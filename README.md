# Subscription Plan UI

A clean and responsive Subscription Plan UI built using HTML and CSS. This project showcases multiple pricing plans with a simple and user-friendly design. It helped me practice Flexbox, CSS styling, layouts, typography, and responsive design techniques while creating an easy-to-understand subscription page interface.

## Features
- Multiple subscription plans
- Clean and modern UI
- Responsive layout
- Easy navigation and readability
- Built with pure HTML and CSS

## Technologies Used
- HTML5
- CSS3

This project is part of my front-end development practice and demonstrates my skills in creating attractive and structured web page layouts.
